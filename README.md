# Sales Trend Analysis Using SQL

## Project Overview

This project analyzes sales data using SQL aggregate functions to identify monthly revenue trends, order volume, profit, and customer insights.

## Objective

- Analyze monthly revenue.
- Calculate monthly order volume.
- Identify top-performing months.
- Analyze revenue by category and state.
- Study payment mode usage.
- Find top customers.

## Tools Used

- MySQL
- SQL
- GitHub

## Dataset

Sales_store_full_data.csv

## SQL Concepts Used

- SELECT
- GROUP BY
- ORDER BY
- SUM()
- COUNT()
- COUNT(DISTINCT)
- AVG()
- ROUND()
- LIMIT
- YEAR()
- MONTH()

## Queries Included

1. Monthly Revenue
2. Monthly Order Volume
3. Average Order Value
4. Highest Revenue Month
5. Top 3 Months
6. Revenue by Category
7. Revenue by State
8. Monthly Profit
9. Quantity Sold
10. Top Customers
11. Payment Mode Analysis

## Outcome

This project demonstrates SQL aggregation techniques for sales trend analysis and business reporting.
